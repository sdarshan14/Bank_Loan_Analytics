# Bank Loan Analytics

## Overview

This project analyzes bank loan application data using Python and exploratory data analysis techniques. The goal is to identify patterns related to loan approvals, applicant demographics, education levels, income distribution, and property areas.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

Dataset: Loan Prediction Dataset

The dataset contains information about:

- Applicant demographics
- Income
- Education
- Property Area
- Loan Amount
- Loan Approval Status
